Calculadora de uma equação do 2° grau

Toda equação na forma ax2+bx+c = 0 com {a, b, c}⊂IR e a ≠ 0, é denominada equação do segundo grau.Onde a, b e c são os coeficientes da equação e x é a incógnita.

Exemplos:
a) Na equação 2x2 + 5x - 2 = 0, temos a = 2, b = 5 e c = - 2.
b) Na equação -3x2 - 2x = 0, temos a = - 3 , b = - 2 e c = 0.
c) Na equação 7x2 - 3 = 0, temos a = 7, b = 0 e c = - 3.
